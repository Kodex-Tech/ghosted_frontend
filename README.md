## Setup

### 1) get dependencies
```bash
flutter pub get
```

### 2) set api url
replace `https://ghosted-api.herokuapp.com` in `lib/pages/home.dart` on `line 117` with your API url.

for example: `https://ghosted-api.herokuapp.com` -> `http://localhost:2500`

### 3) run the website
```bash
flutter run
```
